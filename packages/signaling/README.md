# @rtcio/signaling

A types-only package for use in creating custom signaling for use with the
@rtcio ecosystem.

## Features

- Defines the `ClientSignaler` interface
- Provides types for signaling events

## Installation

```bash
npm i --save-dev @rtcio/signaling
```
